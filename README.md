# 🎬 CineMatch - Movie Recommender System

A modern movie recommendation web application built with Flask, Python, and machine learning.

## 📌 Overview

CineMatch is a movie recommendation web app that recommends similar movies using a content-based filtering approach.  
It features a clean and modern Netflix-style interface where users can search for movies, explore trending content, and view detailed movie information.

## 🚀 Features

- 🔍 Search for movies
- 🎯 Get similar movie recommendations
- 📈 Trending movies section
- 🎭 Genre-based movie categories
- 🖼️ Movie posters display
- ⭐ Movie ratings display
- 🎬 Cast details
- 🎥 Director information
- 🌐 Language information
- ❤️ Favorites section
- 👤 Account / Profile pages
- 💻 Modern Netflix-style UI

## 🛠️ Tech Stack

- **Python**
- **Flask**
- **HTML**
- **CSS**
- **JavaScript**
- **Pandas**
- **Scikit-learn**
- **SQLite**
- **Pickle**

## 🧠 Recommendation Technique

This project uses **Content-Based Filtering** to recommend movies based on similarity between movie metadata such as:

- Genres
- Keywords
- Cast
- Crew
- Overview / Tags
